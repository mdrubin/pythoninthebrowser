
"Python in the Browser" is an interactive Python interpreter that runs inside an HTML textarea. It uses Silverlight 2, and IronPython, and is aimed at providing live examples for tutorials and documentation.

See the 'docs' directory for the licenses and instructions on how to deploy and customize the project.

The project homepage is: http://code.google.com/p/pythoninthebrowser/
You can see an online demo at: http://www.trypython.org/

Using "Python in Your Browser" requires Silverlight 2 to be installed.
